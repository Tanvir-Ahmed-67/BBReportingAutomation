# BBReportingAutomation is a browser automation project which reads data from test.csv file and put all these datas in Bangladesh Banks Reporting System. 
Since BB Reporting system is a tedious form based single row entry system, so to ease bulk rows entry I have automated this task in this project.
How to run this project ? 
 - You need to put your data in test.csv file.
 - Now run the main.java file. Program will start and will put datas to Bangladesh Bank's System one by one from the test.csv file.
 - there is no database involved in this project. So no need any database connection hassle.
